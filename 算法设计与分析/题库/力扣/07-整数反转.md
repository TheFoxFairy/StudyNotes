# 整数反转

```
    public int reverse(int x) {

        if(x > Integer.MAX_VALUE || x < Integer.MIN_VALUE)
            return 0;

        int ans = 0;
        int flag = (x>0?1:-1);

        if(x == 0) return 0;

        x = Math.abs(x);

        while(x>0){
            int temp = x%10;
            if(ans > Integer.MAX_VALUE/10 || ans < Integer.MIN_VALUE/10 ||
                    (ans == Integer.MAX_VALUE/10) && temp > 7 ||
                    (ans == Integer.MIN_VALUE/10) && temp < -8)
                return 0;
            ans = ans*10 + x%10;
            x/=10;
        }
        return ans*flag;
    }
```
